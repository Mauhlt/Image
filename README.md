# Image Loader

## Goals:
1. Create a image library
2. Load image formats:
    - jpg 
    - png 
    - bmp 
    - dib 
    - paint 
    - svg 
    - webp 
    - qoi
    - tga = targa
    - exr
    - dds = direct draw surface
    - dxt
3. Write image formats
4. Convert between formats
5. Benchmark images
    - file size 
    - time to load 
    - quality
